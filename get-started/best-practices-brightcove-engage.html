브라이트코브 인게이지™모범 사례

이 주제에서는 Brightcove Engage로 작업 할 때 권장되는 모범 사례에 대해 알아 봅니다.

비디오 작업

브라이트코브 참여 평가판 계정에는 다음과 같은 제한이 있습니다.

  • 비디오 구성을위한 수동 재생 목록 사용
  • 모바일 / 웹 포털의 컬렉션에 해당 재생 목록을 사용하여 둘 다에 대해 하나의 재생 목록 만 필요하도록합니다.
  • 둘 다에 재생 목록을 추가하여 프로젝트를 쉽게 업데이트하는 방법
  • 각 프로젝트 유형에 대한 이미지 크기
  • 각 색상 가이드
  • 추천 재생 목록을 사용하여 모바일에서 첫 번째 비디오와 웹에서 첫 번째 설정을 모두 수행하는 가장 좋은 방법 — 웹에서 최적의 수… 그것’ s 위의 스레드.
  • 모바일 포털로 시작

 

시작하기

브라이트코브 참여 작업을 시작할 수 있도록 다음 리소스를 활용하는 것이 좋습니다.

 

지원

브라이트코브 참여 평가판과 관련하여 질문이나 문제가 있는 경우브라이트코브 지원에 요청을제출하십시오.
