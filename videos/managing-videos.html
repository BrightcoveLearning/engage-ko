비디오 관리

이 항목에서는 계정에있는 비디오를 관리하는 방법에 대해 알아 봅니다.

계정의 비디오에 액세스하려면브라이트코브 인게이지™클릭비디오탐색 헤더에서.

페이지 본문에는 다음과 같은 내용이 표시됩니다.

  • 비디오 썸네일 및 지속 시간
  • 비디오 이름
  • 비디오 ID - 시스템에 할당된 비디오 ID (모든 계정에서 고유함)
  • 상태
    • - 비디오가 활성 상태임을 나타냅니다.
    • - 동영상이 비활성 상태임을 나타냅니다 (재생할 수 없음)
  • 참조 ID
  • 생성된 날짜 및 시간 및 사용자 ID
  • 업데이트된 날짜 및 시간 및 사용자 ID

동영상 썸네일 옆에 있는 체크박스를 클릭한 다음 Shift 키를 누른 채 다른 체크박스를 클릭하면 여러 동영상을 선택할 수 있습니다. 열 머리글을 클릭하여 비디오 목록을 정렬할 수 있습니다. 동영상 페이지는 “고정” 정렬을 사용하며 정렬 선택을 기억합니다. 탐색하고 다시 돌아오거나 다시 로그인하면 비디오 페이지가 마지막으로 선택한 정렬로 기본 설정됩니다.

페이지 상단의 버튼 막대는 다음을 수행할 수 있는 기능을 제공합니다.

  • 업로드 -계정에 동영상 업로드
  • 빠른 편집 -빠른 편집 비디오 속성
  • 퀵뷰 -빠르게 비디오 속성보기
  • 더 -에 비디오 추가재생 목록 , 활성화 / 비활성화비디오 및동영상 공유다른 계정에
  • 지우다 -계정에서 동영상 제거

비디오 속성 편집

비디오 속성을 편집하려면 비디오 이름을 클릭하여 비디오 속성 페이지를 엽니다. 보다비디오 속성 편집비디오 속성 편집에 대한 정보는

그만큼빠른 편집기능은 여러 비디오의 속성을 편집 할 때 유용합니다.

동영상 업로드

계정에 비디오를 업로드하는 방법에 대한 자세한 내용은비디오 업로드 .

빠른 편집 사용

그만큼빠른 편집기능은 여러 비디오의 속성을 편집 할 때 유용합니다. 자세한 내용은빠른 편집을 사용하여 비디오 속성 편집 .

빠른보기 사용

빠른보기 패널을 사용하여 비디오 속성을 빠르게 볼 수 있습니다. 빠른 보기 패널을 열고 닫으려면빠른 보기를 클릭합니다 . 패널에는 선택한 비디오에 대한 비디오 속성의 하위 집합이 표시됩니다.

빠른보기 패널은 읽기 전용입니다. 여러 비디오를 선택한 경우 빠른보기에 마지막으로 선택한 비디오의 속성이 표시됩니다.

재생 목록에 비디오 추가

재생 목록은 플레이어에서 재생하기 위해 함께 그룹화 된 비디오 모음입니다. 재생 목록을 만든 후 Brightcove Engage 응용 프로그램을 만들 때 사용할 수 있습니다. 재생 목록 생성 및 관리에 대한 자세한 내용은재생 목록 생성 및 관리 .

비디오 활성화 및 비활성화

비디오 페이지의상태열은 비디오가활성화또는비활성화되었는지 여부를 나타냅니다 .

  • - 비디오가 활성 상태임을 나타냅니다 (비디오를 재생할 수 있음).
  • - 동영상이 비활성 상태임을 나타냅니다 (비디오를 재생할 수 없음)

비디오를활성화하려면 비디오를 선택한 다음자세히 > 활성화를 클릭합니다 . 비디오를비활성화하려면 비디오를 선택한 다음자세히 > 비활성화를 클릭합니다 . 상태열이 표시되지 않으면 열 선택기를 사용합니다. 를 클릭하여 선택합니다.

다른 계정에 비디오 공유

미디어 공유를 통해 게시자는 동영상을 한 번 업로드한 다음 다른 계정에 동영상을 공유할 수 있습니다. 계정은 본인, 회사 계정 또는 다른 게시자의 계정일 수 있습니다. 동영상 섬네일은 동영상이 계정에 공유되었는지 또는 다른 계정에 공유되었는지를 나타냅니다. 아래 예에서 CEO 환영동영상이 다른 계정에 공유되었으며 Brightcove 개요동영상이이 계정으로 공유되었습니다.

다른 계정에 비디오를 공유하는 방법에 대한 자세한 내용은공유 비디오 작업 .

비디오 삭제하기

비디오를 삭제하려면 삭제할 비디오의 비디오 축소판 옆에 있는 확인란을 클릭한 다음 삭제 버튼을 클릭합니다 (). 를 클릭하여 삭제를 확인합니다. 삭제단추.

생성 및 업데이트된 날짜 및 시간 정보

동영상을 생성 및/또는 업데이트한 사용자의 사용자 ID가 [생성 날짜/업데이트 날짜] 열에 표시됩니다. 작업을 수행한 사용자가 계정에서 제거된 경우에도 해당 ID는 계속 표시됩니다.

동영상을 변경해도 사용자 ID와 사용자 ID가 표시되지 않는 경우가 기록되지 않는 경우가 있습니다.

  • 비디오 소스가 교체되면 비디오의 업데이트된 날짜가 업데이트되지만 작업을 시작한 사용자는 기록되지 않습니다.
  • 이미지 (포스터/썸네일) 및/또는 텍스트 트랙이 업데이트되면 비디오의 업데이트 날짜가 업데이트되지만 작업을 시작한 사용자는 기록되지 않습니다.
  • 비디오가 제휴 계정에 공유되면 생성 된 날짜에 관련 사용자 ID가 표시되지 않습니다.

동영상 검색

검색 필드는 계정의 동영상 및 재생 목록을 검색하는 데 사용할 수 있습니다. 자세한 내용은비디오 검색 방법 .

비디오 데이터 내보내기

딸깍 하는 소리수출모든 비디오 또는 비디오 하위 집합에 대한 비디오 메타 데이터를 .csv 파일로 내 보냅니다. 자세한 내용은비디오 데이터 내보내기 .
