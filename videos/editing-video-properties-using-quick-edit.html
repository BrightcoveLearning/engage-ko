빠른 편집을 사용하여 비디오 속성 편집

이 항목에서는 빠른 편집 기능을 사용하여 비디오 속성을 편집하는 방법에 대해 알아 봅니다.
 

빠른 편집 기능은 여러 비디오의 비디오 속성을 편집하는 편리한 방법을 제공합니다. 확인란을 클릭 한 다음 다른 확인란을 Shift- 클릭하여 여러 비디오를 선택할 수 있습니다 (Shift 키를 누른 상태에서 두 번째 비디오를 클릭하면 두 비디오와 그 사이의 모든 비디오가 선택됩니다). 클릭빠른 편집비디오 페이지 오른쪽에 빠른 편집 패널이 열립니다.

빠른 편집 패널에는 비디오 속성의 하위 집합이 표시됩니다.

  • 이름
  • 간단한 설명
  • 장황한 설명
  • 추가 할 태그
  • 제거 할 태그
  • 관련 링크 URL
  • 관련 링크 텍스트
  • 시작 / 종료 날짜 예약
  • 사용자 지정 필드
참고: 이러한 모든 속성에 대한 설명은 다음을 참조하십시오. 비디오 속성 편집문서.

빠른 편집 패널을 사용하여 값을 지정하십시오. 완료되면저장변경 사항을 저장하거나취소나가기 위해서.

빠른 편집 사용에 대한 참고 사항

  • 하나 이상의 비디오에 속성에 할당된 값이 있는 경우 혼합값편집이 표시됩니다.
  • 속성 이름 뒤의 별표 (*)는 필수 속성을 나타냅니다.
  • 등록 정보 변경 사항을 취소하려면복원을 클릭합니다.
