비디오에 텍스트 트랙 추가

이 항목에서는 비디오에 텍스트 트랙을 추가하는 방법을 배웁니다.

텍스트 트랙 속성은 비디오에 캡션을 제공하도록 구성할 수 있습니다. 캡션은 액세스하려는 개인에게 추가 또는 해석 정보를 제공합니다. 청각 장애인과 청각 장애인이 비디오 프레젠테이션의 오디오 부분에 액세스 할 수 있도록 자막이 필요합니다. 비디오에 대한 캡션을 표시하려면 다음 작업을 완료해야 합니다.

  • 캡션 파일 만들기
  • 텍스트 트랙 속성 구성

캡션 파일 만들기

비디오에 캡션을 추가하려면 먼저 캡션 파일을 만들어야 합니다. WebVTT (Web Video Text Tracks Format), SRT 및 DFXP 형식이 지원됩니다. WebVTT 파일 형식 (아래 표시) 은 일반 텍스트이며 시간 텍스트 데이터 다음에 원하는 캡션 또는 자막 텍스트가 포함됩니다.

중간 선: 90% 00:00:07 .080 정렬: 중간 선: 90% 00:00:07 .080 —> 00:00:10 .280 정렬:중간 선: 84% 안녕하세요, 저는 브라이트코브의 학습 전문가인 밥 베일리입니다. 00:00:10.280 --> 00:00:12.780 align:middle line:84% 이 비디오에서는 Brightcove Engage가 00:00:12.780 --> 00:00:14.940 align:middle line:84% 구성 및 간략한 개요 보기 00:00:14.940 --> 00:00:17.490 align:middle line:90% 당신은 그것을 사용하여 달성할 수 있습니다.
참고: 캡션 파일 이름에는 문자 (대문자 또는 소문자), 숫자, 대시 (-) 및 밑줄 (_) 만 포함되어야 합니다.

WebVTT 형식에 대한 자세한 내용은이 링크를 참조하십시오 . 지원해야 하는 각 언어에 대해 별도의 캡션 파일이 있어야 합니다. 캡션 파일을 만드는 데 도움이 되는 타사 도구가 많이 있지만 브라이트코브는 전사, 캡션 및 번역 서비스를 제공하는 여러 파트너를 보유하고 있습니다. 브라이트코브 파트너의 전체 목록은 당사 웹사이트의파트너섹션을 참조하십시오.

텍스트 트랙 속성 구성

캡션 파일이 생성되면 브라이트코브에서 업로드하고 호스팅하거나 인터넷에 액세스할 수 있는 URL에 상주할 수 있습니다.

비디오의 텍스트 트랙 속성을 보고 편집하려면 다음 단계를 수행하십시오.

  1. 비디오페이지를 엽니다.
  2. 비디오 제목 링크를 클릭하여 비디오 속성 페이지를 엽니다.
  3. 텍스트 트랙섹션을 찾아편집을클릭하거나 섹션을 두 번 클릭합니다.
  4. 텍스트 트랙 파일을 Brightcove에 업로드할지 아니면 외부 URL에서 호스팅할지 선택합니다.
    • 파일 업로드 - 찾아보기를클릭하고 파일 시스템에서 텍스트 트랙 파일을 선택합니다. 파일이 브라이트코브에 업로드됩니다.
    • URL 업로드 - 텍스트 트랙 파일이 있는 공용 URL을 입력한 다음업로드를 클릭합니다 . 파일이 브라이트코브에 업로드됩니다.
    • 원격 URL - 텍스트 트랙 파일이 있는 공용 URL을 입력합니다. URL은 250자 미만이어야 합니다. 원격 URL에서 캡션이 로드됩니다.
      참고: 자체 서버에서 캡션 파일을 호스팅하는 경우 파일이 공개적으로 액세스 할 수있는 URL에 있는지 확인하십시오. 캡션 파일을 호스팅하는 서버에도 CORS (교차 출처 리소스 공유) 가 활성화되어 있어야 합니다.
      메모: 괄호, 물음표, 따옴표, 공백 및 기타 특수 문자가 포함된 텍스트 트랙 파일 이름은 업로드할 때 자동으로 실패합니다. 브라이트코브는 특수 문자 없이 파일 이름을 사용할 것을 권장합니다.
      참고: 계정에 공유된 동영상의 텍스트 트랙은 수정할 수 없습니다.
  5. NS 언어필드에는 다음이 포함되어야 합니다. 두 글자 언어 코드트랙 텍스트 데이터의 언어를 지정합니다.
  6. 적절한 입력상표캡션 파일의 경우. 레이블은 비디오 플레이어 내부에 표시되며 시청자가 언어를 선택할 수 있도록합니다. 레이블이 제공되지 않으면 언어 코드가 표시됩니다.
  7. 텍스트종류 트랙을 선택합니다. 옵션에는 다음이 포함됩니다.
    • 캡션 - 자막과 유사하지만 음향 효과 및 기타 오디오 정보도 포함될 수 있습니다.
    • 자막 - 대화의 전사 또는 번역
    • 설명 - 화면 판독기를 통해 비디오를 설명하는 별도의 텍스트 파일입니다.
    • 챕터 - 사용자가 비디오를 탐색하는 데 도움이 됩니다.
    • 메타데이터 - 기본적으로 뷰어에게 표시되지 않는 비디오에 대한 정보 및 콘텐츠 (JavaScript를 사용하여 표시하고자 할 수도 있음)
  8. 텍스트 트랙 파일의 MIME 유형을입력합니다. 기본값을선택하면 비디오가 재생될 때 캡션이 표시됩니다.
  9. 기본 (자동 표시)을 선택하여 트랙을 기본값으로 설정합니다. 대부분의 브라우저는 기본 텍스트 트랙을 먼저 표시하려고 시도하지만 언어 및 접근성 설정이 기본 설정을 무시할 수 있습니다.
    참고: 기본 설정은 IE11에서 작동하지 않습니다.
  10. 텍스트 트랙추가를 클릭하여 텍스트 트랙 정보를 저장합니다.
  11. 텍스트 트랙추가를 클릭하여 다른 텍스트 트랙을 추가하거나저장을클릭하여 텍스트 트랙을 저장합니다.
    참고: 텍스트 트랙 속성에 대한 변경 사항이 나타나는 데 최대 15분이 걸릴 수 있습니다.

텍스트 트랙이 기본값으로 지정된 경우굵게 표시됩니다 . 텍스트 트랙을 삭제하려면 삭제 아이콘 () 텍스트 트랙 옆에 있습니다.

텍스트 트랙이있는 동영상은 플레이어 크롬에 CC 버튼이 표시됩니다. CC버튼을 클릭하여 캡션 언어를 선택하고 캡션을 표시합니다.

제한 사항

캡션은 다음 장치 유형 및 운영 체제에서 지원됩니다.

  • 아이폰 OS 10+
  • 안드로이드 7+
  • 인터넷 익스플로러 11+
  • 데스크톱의 최신 크롬, 사파리 및 파이어폭스 브라우저
  • 캡션 길이에는 기술적 제한이 없지만 전체 캡션을 표시하려면 플레이어가 많은 텍스트를 표시할 수 있을 만큼 커야 합니다.
