웹 경험 페이지에 이미지, 텍스트 및 링크 추가

이 주제에서는 웹 경험 페이지에 이미지, 텍스트 및 링크를 추가하는 방법을 학습합니다.

이미지, 텍스트 및 링크와 같은 개체를 Web Experience 페이지에 추가하여 비디오 경험을 사용자 정의하고 브랜딩 할 수 있습니다.

페이지에 개체 추가

경험 편집기를 사용하여 이미지, 텍스트 및 링크와 같은 개체를 페이지에 추가 할 수 있습니다. 페이지를 편집하려면 환경을 편집한 다음페이지왼쪽 탐색에서 페이지 템플릿은 페이지에 자리 표시자와 연필 아이콘()는 편집 가능한 개체를 나타냅니다. 이 자리 표시자를 사용하여 페이지에 이미지, 텍스트 및 링크를 추가할 수 있습니다.

페이지에 텍스트 추가

몇 가지 다른 방법으로 페이지에 텍스트를 추가 할 수 있습니다.

  • 클릭하여 + 콘텐츠 추가페이지 상단의 링크
  • 페이지에서 텍스트 자리 표시자를 클릭하여

클릭하여 텍스트 추가 + 콘텐츠 추가링크

페이지에 텍스트를 추가하려면 + 콘텐츠 추가링크:

  1. 클릭 + 콘텐츠 추가링크.
  2. 고르다본문그런 다음저장 .
  3. 클릭자리 표시 자페이지에 추가되고 텍스트를 입력합니다. 딸깍 하는 소리구하다완료되면.

텍스트 자리 표시자를 클릭하여 텍스트 추가

페이지에는 사용자 지정 텍스트를 페이지에 추가 할 수있는 수많은 텍스트 개체가 포함됩니다. 연필 아이콘은 요소를 편집 할 수 있음을 나타냅니다.

텍스트를 추가하려면 다음과 같이 하십시오.

  1. 자리 표시 자 개체 중 하나를 클릭합니다.
  2. 텍스트를 입력하고저장을 클릭합니다 .

페이지 헤더에 이미지 추가

템플릿에 따라 이미지 크기가 조정됩니다 (확대되지 않음). 이미지 업 로더는 이미지가 사용되는 위치에 따라 권장 이미지 크기를 표시 할 수 있습니다. JPEG, PNG 및 GIF 이미지가 지원됩니다. 페이지 머리글에 이미지를 추가하려면 다음과 같이 하십시오.

  1. 클릭 + 콘텐츠 추가링크.
  2. 고르다업로드그런 다음저장 .
  3. 클릭 + 사진 추가링크. 이미지 브라우저 대화 상자가 열립니다.
  4. 새 이미지를 업로드하려면파일을 선택버튼을 클릭 한 다음 파일 시스템에서 이미지를 찾아 선택합니다. 드래그 앤 드롭 및 이미지도 가능합니다. 딸깍 하는 소리구하다완료되면.
    참고: 브라이트코브는 이미지의 크기를 140px x 50px로 조정합니다.

페이지 바닥 글에 탐색 링크 추가

페이지 바닥 글에 탐색 링크를 추가 할 수 있습니다. 이것은 제목과 아래의 일련의 링크를 원할 때 유용합니다. 예를 들면 다음과 같습니다.

페이지 바닥 글 탐색 개체는 두 부분으로 구성됩니다. 자리 표시 자제목과 + 더 추가링크.

페이지 바닥 글에 탐색 링크를 추가하려면 :

  1. 딸깍 하는 소리자리 표시 자헤더 값을 입력합니다.
  2. 딸깍 하는 소리 + 더 추가그리고 입력본문과링크 . 링크가 작성될 때목록 순서값을 설정하여 링크의 순서를 지정할 수도 있습니다.
  3. 저장을 클릭합니다 .

추가 탐색 링크는 클릭하여 입력 할 수 있습니다. + 더 추가 .

페이지에서 개체 삭제

텍스트, 이미지 및 링크는 몇 가지 다른 방법으로 페이지에서 제거 할 수 있습니다.

페이지 머리글에서 텍스트 또는 이미지를 제거하려면

  1. 텍스트 또는 이미지와 관련된 페이지 헤더에서 추적 화살표 아이콘을 클릭합니다.

페이지 바닥 글에서 탐색 링크를 제거하려면

  1. 페이지 바닥 글에있는 링크를 클릭하십시오.
  2. 딸깍 하는 소리없애다 .

페이지 바닥 글에서 제목을 제거하려면

  1. 페이지 바닥 글의 제목을 클릭하십시오.
  2. 제목 텍스트를 지우고저장을 클릭합니다 .
