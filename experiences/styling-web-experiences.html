웹 경험의 스타일 사용자 정의

이 주제에서는 웹 경험의 스타일을 사용자 정의하는 방법을 학습합니다.

경험 편집기를 사용하여 테마, 색상 및 글꼴을 변경하여 웹 경험의 스타일을 사용자 정의 할 수 있습니다. 스타일을 편집하려면 경험을 편집한 다음스타일왼쪽 탐색에서

테마 선택하기

선택테마사전 정의 된 글꼴 및 색상 세트를 경험에 적용합니다. 선택하려면테마 :

  1. 딸깍 하는 소리스타일왼쪽 탐색에서
  2. 선택주제드롭다운 목록에서. 테마를 선택하면 테마를 미리 볼 수 있도록 환경이 새로 고쳐집니다.
  3. 경험 편집기로 돌아가려면 경험으로 < 돌아가기를 클릭합니다 .

색상 사용자 정의

색상설정은 테마에서 제공하는 색상을 미세 조정하는 데 사용할 수 있습니다. 예를 들어 색상 설정을 사용하여 배경 및 글꼴 색상을 미세 조정할 수 있습니다. 색상을사용자 정의하려면

  1. 왼쪽 탐색에서 STYLE아이콘을 클릭합니다. 색상섹션에는 모든 페이지 객체와 해당 색상이 표시됩니다.
  2. 새 색상을 선택하려면 색상 원을 클릭하십시오.
  3. 색상 선택기를 사용하여 색상 사각형의 색상을 클릭하거나 색상 사각형 아래에 16진수 색상 번호를 입력하여 색상을 선택합니다. 색상 사각형 아래의 색상 스펙트럼을 클릭하여 색상 범위를 변경할 수 있습니다. X를 클릭하여 색상 피커를 닫고 변경 사항을 저장합니다.

    색상 선택을 재설정하려면 색상 원의 왼쪽에 있는 왼쪽 화살표를 클릭합니다.

  4. 경험 편집기로 돌아가려면 경험으로 < 돌아가기를 클릭합니다 .

글꼴 선택

경험을 위해 다른 글꼴을 선택하려면 :

  1. 딸깍 하는 소리스타일왼쪽 탐색에서
  2. 새로 선택글꼴드롭다운 목록을 사용합니다.

    글꼴 선택을 재설정하려면 글꼴 드롭다운 위에 있는재설정링크를 클릭합니다.

  3. 경험 편집기로 돌아가려면 경험으로 < 돌아가기를 클릭합니다 .
