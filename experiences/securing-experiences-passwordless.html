Auth0 암호없는 인증을 사용하여 경험 보호

이 항목에서는 Auth0 암호없는 인증을 사용하여 환경을 보호하는 방법을 알아 봅니다.

비밀번호없는 인증으로 Auth0을 사용하여 경험을 보호 할 수 있습니다. 이 보안 방법을 사용하는 경우 사용자는 전화 번호를 입력해야합니다. Auth0은 다음을 사용하여 일회용 코드를 보냅니다. 트윌리오 . 그런 다음 사용자는 애플리케이션에 코드를 입력합니다. 코드에 첨부 된 전화 번호가 기존 사용자와 일치하면 Auth0이 사용자를 인증합니다.

참고: 이 항목의 단계 및 이미지는 평가판을 사용하고 있습니다. NS 인증0그리고트윌리오 .

시작하기 전에

Auth0 Passwordless를 구성하기 전에 다음 태스크를 완료하십시오.

  • 에 설명 된대로 Auth0을 사용하여 모바일 앱 환경 구성 Auth0를 사용하여 경험 보호
  • 가입하기 Twilio평가판 계정 및 계정 확인

Twilio 전화 번호 받기

Passwordless를 구성 할 때 발신자 번호로 사용할 Twilio 전화 번호를 받아야합니다.

  1. Twilio 계정에 로그인합니다.
  2. 에계기반페이지, 클릭평가판 번호 받기 .
  3. 딸깍 하는 소리이 번호 선택 .
  4. 딸깍 하는 소리끝난번호를 확인합니다.

Auth0에서 암호없는 연결 구성

Auth0에서 암호없는 연결을 구성하여 시작하십시오.

  1. Auth0 계정에 로그인합니다.
  2. 왼쪽 탐색 창에서연결> 암호 없음 .
  3. 활성화 SMS연결.

Twilio SID 입력

  1. 트빌리오로 돌아가세요.
  2. 대시 보드 페이지에서계정 SID들.
  3. 값을 클립보드로 복사합니다.
  4. Auth0으로 돌아갑니다.
  5. 값을 Twilio SID들.

Twilio 인증 토큰 입력

  1. 트빌리오로 돌아가세요.
  2. 대시 보드 페이지에서인증 토큰들.
  3. 클릭보여 주다토큰을 공개하는 링크.
  4. 값을 클립보드로 복사합니다.
  5. Auth0으로 돌아갑니다.
  6. 값을 Twilio 인증 토큰들.

Twilio From 전화 번호 입력

  1. 트빌리오로 돌아가세요.
  2. 대시 보드 페이지에서시험 번호당신이전에 생성 .
  3. 번호를 클립 보드에 복사합니다.
  4. Auth0으로 돌아갑니다.
  5. 값을에서들.
  6. (선택 사항) 메시지본문.

애플리케이션 활성화

  1. 클릭응용대화 상자 상단의 링크.
  2. Brightcove Engage 애플리케이션을 보호하기 위해 생성 한 Auth0 애플리케이션을 활성화합니다.
  3. 딸깍 하는 소리저장대화 상자를 닫습니다.

비밀번호없는 로그인을 사용하도록 애플리케이션 구성

  1. 왼쪽 탐색 창에서범용 로그인 .
  2. 클릭로그인링크.
  3. 활성화로그인 페이지 사용자 지정선택권.
  4. 고르다잠금 (비밀번호 없음) ~로부터기본 템플릿드롭 다운 목록.
  5. 딸깍 하는 소리변경 사항을 저장하다 .

경험 미리 보기

위의 단계가 완료되면 사용자 경험을 미리보고 뷰어에게 전화 번호를 입력하라는 메시지가 표시되는지 확인할 수 있습니다.

  1. 브라이트코브 참여에 로그인합니다.
  2. Auth0로 보호 된 경험 제목을 클릭하십시오.
  3. 경험을 미리 봅니다. 전화 번호를 묻는 로그인 화면이 표시되어야합니다.
  4. 미리보기를 닫습니다.

 

Twilio에 유효한 전화 번호 추가

테스트 목적으로 Twilio에 하나 이상의 전화 번호를 추가합니다.

  1. 트빌리오로 돌아가세요.
  2. 왼쪽 탐색 창에서모든 제품 및 서비스상.
  3. 왼쪽 탐색 창에서전화 번호 .
  4. 왼쪽 탐색 창에서확인 된 호출 ID .
  5. 테스트에 사용할 번호를 입력하십시오. 평가판 계정에는 평가판 계정을 만들 때 사용한 전화 번호가 있어야합니다.

구성 검증

마지막 단계는 시청자가 휴대폰 번호를 사용하여 로그인 할 수 있는지 테스트하는 것입니다.

  1. 장치에서 Brightcove Engage 애플리케이션을 엽니 다. 메시지가 표시되면액세스 코드당신의 경험을 위해.
  2. 전화 번호를 입력하고제출 .
  3. 인증 코드가 포함 된 SMS 메시지를 받았는지 확인합니다.
  4. 인증 코드를 입력하고제출 .

Brightcove Engage 모바일 앱이 표시되어야합니다.
