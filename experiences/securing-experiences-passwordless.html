    Auth0 암호 없는 인증을 사용하여 환경 보호

    이 항목에서는 Auth0 암호 없는 인증을 사용하여 환경을 보호하는 방법에 대해 설명합니다.

    환경은 암호 없는 인증과 Auth0을 사용하여 보안을 유지할 수 있습니다. 이 보안 방법을 사용하면 사용자에게 전화 번호를 입력하라는 메시지가 표시됩니다. Auth0는 다음을 사용하여 일회용 코드를 보냅니다.   Twilio . 그런 다음 사용자는 응용 프로그램에 코드를 입력합니다. 코드에 첨부된 전화 번호가 기존 사용자와 일치하면 Auth0이 사용자를 인증합니다.

    참고: 이 항목의 단계와 이미지는 평가판을 사용하고 있습니다.의 Auth0과 Twilio .

    시작하기 전에

    Auth0 암호를 사용하지 않도록 구성하기 전에 다음 작업을 완료하십시오.

    • 에 설명 된대로 Auth0을 사용하여 모바일 앱 환경 구성 Auth0를 사용하여 경험 보호
    • 가입하기 Twilio평가판 계정 및 계정 확인

    Twilio 전화 번호 받기

    Passwordless를 구성할 때 발신자 번호로 사용되는 Twilio 전화 번호를 받아야 합니다.

    1. Twilio 계정에 로그인합니다.
    2. 에계기반페이지, 클릭평가판 번호 받기 .
    3. 딸깍 하는 소리이 번호 선택 .
    4. 딸깍 하는 소리끝난번호를 확인합니다.

    Auth0에서 암호 없는 연결 구성

    먼저 Auth0에서 암호 없는 연결을 구성합니다.

    1. Auth0 계정에 로그인합니다.
    2. 왼쪽 탐색 창에서연결> 암호 없음 .
    3. 활성화 SMS연결.

    트와일리오 SID 입력

    1. 트와일리오로 돌아가
    2. 대시 보드 페이지에서계정 SID들.
    3. 값을 클립보드에 복사합니다.
    4. Auth0으로 돌아갑니다.
    5. 값을 Twilio SID들.

    트와일리오 인증 토큰 입력

    1. 트와일리오로 돌아가
    2. 대시 보드 페이지에서인증 토큰들.
    3. 클릭보여 주다토큰을 공개하는 링크.
    4. 값을 클립보드에 복사합니다.
    5. Auth0으로 돌아갑니다.
    6. 값을 Twilio 인증 토큰들.

    Twilio 전화 번호 입력

    1. 트와일리오로 돌아가
    2. 대시 보드 페이지에서시험 번호당신이전에 생성 .
    3. 클립보드에 번호를 복사합니다.
    4. Auth0으로 돌아갑니다.
    5. 값을에서들.
    6. (선택 사항) 메시지본문.

    응용 프로그램 활성화

    1. 클릭응용대화 상자 상단의 링크.
    2. 생성한 Auth0 애플리케이션을 활성화하여 브라이트코브 Engage 애플리케이션을 보호합니다.
    3. 딸깍 하는 소리저장대화 상자를 닫습니다.

    암호 없는 로그인을 사용하도록 응용 프로그램 구성

    1. 왼쪽 탐색 창에서범용 로그인 .
    2. 클릭로그인링크.
    3. 활성화로그인 페이지 사용자 지정선택권.
    4. 고르다잠금 (비밀번호 없음) ~로부터기본 템플릿드롭 다운 목록.
    5. 딸깍 하는 소리변경 사항을 저장하다 .

    경험 미리 보기

    위의 단계가 완료되면 환경을 미리 보고 시청자에게 전화번호를 입력하라는 메시지가 표시되는지 확인할 수 있습니다.

    1. 브라이트코브 Enage에 로그인합니다.
    2. Auth0으로 보안이 설정된 환경 타이틀을 클릭합니다.
    3. 경험을 미리 봅니다. 전화 번호를 묻는 로그인 화면이 나타납니다.
    4. 미리보기를 닫습니다.

     

    Twilio에 유효한 전화번호 추가하기

    테스트 목적으로 하나 이상의 전화번호를 Twilio에 추가합니다.

    1. 트와일리오로 돌아가
    2. 왼쪽 탐색 창에서모든 제품 및 서비스상.
    3. 왼쪽 탐색 창에서전화 번호 .
    4. 왼쪽 탐색 창에서확인 된 호출 ID .
    5. 테스트에 사용할 숫자를 입력합니다. 평가판 계정에는 평가판 계정을 만들 때 사용한 전화번호가 있어야 합니다.

    구성 유효성 검사

    마지막 단계는 시청자가 휴대 전화 번호를 사용하여 로그인 할 수 있는지 테스트하는 것입니다.

    1. 디바이스에서 브라이트코브 Engage 애플리케이션을 엽니다. 메시지가 표시되면액세스 코드당신의 경험을 위해.
    2. 전화 번호를 입력하고제출 .
    3. 확인 코드가 포함된 SMS 메시지를 수신하는지 확인합니다.
    4. 인증 코드를 입력하고제출 .

    브라이트코브 Engage 모바일 앱이 표시됩니다.
