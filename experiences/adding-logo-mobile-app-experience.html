모바일 앱 경험에 로고 추가

이 항목에서는 모바일 앱 경험 상단에 로고를 추가하는 방법에 대해 알아 봅니다.

경험 편집기를 사용하여 모바일 앱 경험의 상단에 로고를 추가 할 수 있습니다. 페이지를 편집하려면 환경을 편집한 다음페이지왼쪽 탐색에서 페이지 템플릿의 미리보기가 나타납니다. 그만큼로고 추가링크는 로고가 나타날 위치를 나타냅니다.

페이지에 로고 추가

이미지 크기가 조정됩니다 (확대되지 않음). JPEG, PNG 및 GIF 이미지가 지원됩니다. 페이지 머리글에 로고를 추가하려면 :

  1. 클릭로고 추가링크. 로고 추가 대화 상자가 열립니다.
  2. 딸깍 하는 소리검색이미지를 찾아 선택합니다. 드래그 앤 드롭하여 이미지를 만들거나 이미지 파일에 대한 링크를 입력 할 수도 있습니다. 딸깍 하는 소리구하다완료되면.
    참고: 브라이트코브는 이미지 크기를 65px x 30px로 조정합니다.
    참고: 브라이트코브는 < 20KB 크기의 이미지를 사용할 것을 권장합니다.
  3. 이미지가 앱 미리보기에 표시됩니다.

로고를 편집하거나 삭제하려면 로고 위로 마우스를 가져간 다음 해당 아이콘을 클릭하십시오.
