경험에 비디오 추가

이 항목에서는 경험에 비디오를 추가하는 방법을 배웁니다.

컬렉션 편집기를 사용하여 컬렉션을 만들고 경험에 표시 할 비디오를 선택할 수 있습니다. 경험을 통해 비디오는 컬렉션으로 구성됩니다.

컬렉션 만들기

컬렉션을 만들려면 다음 단계를 수행합니다.

  1. 딸깍 하는 소리비디오왼쪽 탐색에서 컬렉션 관리 페이지가 열립니다. 페이지 오른쪽에는 선택한 템플릿 유형에 대한 홈 페이지 비디오 레이아웃의 미리보기가 표시됩니다.
  2. 컬렉션의 이름을 입력하고 Enter 키를 누릅니다 . 이 이름은 시청자에게 표시되므로 의미 있는 이름을 사용해야 합니다. 편집을 클릭합니다 () 아이콘을 사용하여 컬렉션 이름을 편집합니다.

모바일 앱 경험 작업

모바일 앱 경험을 만들 때 첫 번째 컬렉션의 첫 번째 비디오가 추천 비디오로 사용됩니다. 추천 동영상은 응용 프로그램 상단에 나타납니다.

모바일 앱 경험용 컬렉션을 만들 때 브라이트코브는추천 비디오컬렉션을 첫 번째 컬렉션으로 만들 것을 권장합니다. 할당함으로써수동 재생 목록추천 동영상 컬렉션에서 추천 동영상은 다음을 사용하여 수동 재생 목록을 편집하여 쉽게 변경할 수 있습니다. 동영상페이지. 또한 응용 프로그램을 편집하고 다시 게시 할 필요없이 추천 비디오를 변경할 수 있습니다.

컬렉션 편집

  1. 컬렉션을 편집하려면 컬렉션 이름을 클릭합니다.
  2. 컬렉션 편집 페이지가 열립니다. 컬렉션 편집 페이지는 컬렉션에 비디오를 추가하고 추가 컬렉션 속성을 구성하는 데 사용됩니다. 컬렉션변경드롭다운에는 생성된 모든 컬렉션이 나열됩니다. 편집할 컬렉션을 선택합니다.
  3. 컬렉션 소스는 컬렉션에 대한 비디오 콘텐츠의 소스를지정합니다.
    • Brightcove -Brightcove 계정의 비디오가 사용됩니다.
  4. Brightcove 컬렉션 소스를 사용하면 비디오를재생 목록또는 사용커스텀할당:
    • 재생 목록 -컬렉션의 비디오는 재생 목록을 기반으로합니다.
    • 커스텀 -컬렉션의 비디오는 비디오 태그, 사용자 지정 검색 또는 사용자 지정 메타 데이터 필드를 기반으로합니다.
    참고: 최근에 비디오 메타데이터 또는 재생 목록을 변경한 경우 해당 변경 사항이 컬렉션 편집기 및 게시된 환경에 표시되는 데 몇 분 정도 걸릴 수 있습니다.

    재생 목록 비디오 할당 사용

    사용하기로 선택한 경우재생 목록비디오 할당, 컬렉션의 비디오는 재생 목록을 기반으로합니다.

    1. 재생 목록옵션을 선택합니다. 계정의 재생 목록 목록이 나타납니다.
    2. 탐색 컨트롤() 또는 검색 기능을 사용하여 원하는 재생 목록을 찾습니다. 재생 목록을 선택하면 해당 재생 목록의 비디오 목록이 표시됩니다.
      참고: 컬렉션의 비디오 순서는 재생 목록 설정에 따라 결정됩니다.
    3. 저장을클릭하여 변경 사항을 저장합니다.

    사용자 지정 비디오 할당 사용

    사용자 지정비디오 할당을 사용하도록 선택하면 컬렉션의 동영상은 비디오 태그, 사용자 지정 검색 또는 사용자 지정 메타데이터 필드를 기반으로 합니다.

    1. 사용자 지정옵션을 선택합니다.
    2. 검색 매개 변수는태그 , 검색을 기반으로비디오포함또는제외로 설정할 수 있습니다.또는사용자 정의 필드 .
      참고: 사용할 때사용자 정의 필드옵션, 당신은 입력해야내부의사용자 정의 필드의 이름, 아니다 NS 표시하다이름.
    3. 원하는 비디오를 반환하려면 적절한 검색 매개 변수를 구성하십시오. 다음 예에서 태그가 지정된 동영상은영업 교육반환됩니다.
    4. (선택 사항) 추가 아이콘() 추가 매개변수를 포함하거나 제외하려면 해당 섹션에 입력합니다.
    5. 비디오 정렬 순서 설정:
      • 가장 최근 -활성화 된시기를 기준으로 정렬 된 동영상, 가장 최근 동영상이 먼저 나열 됨
      • 오래된 것 먼저 -활성화 된시기를 기준으로 정렬 된 동영상, 가장 오래된 동영상이 먼저 나열 됨
      • 가장 많이 본 -대부분의 모든 시간보기
      • AZ -오름차순으로 정렬
      • ZA -내림차순으로 정렬
    6. 저장을클릭하여 변경 사항을 저장합니다.
