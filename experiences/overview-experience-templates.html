경험 템플릿 개요

이 항목에서는 사용 가능한 다양한 경험 템플릿에 대해 알아 봅니다.

경험 템플릿은 경험의 전체적인 모양과 느낌을 제어합니다. 새 경험을 만들 때 경험에 사용할 템플릿을 선택합니다. 템플릿을 선택한 후에는 템플릿을 변경할 수 없습니다.

다음 템플릿을 사용할 수 있습니다.

  • 모바일 앱
  • 웹 템플릿

각 템플릿은 경험 편집기에서 사용자 지정 및 / 또는 미리 볼 수있는 페이지 템플릿 집합을 제공합니다. 페이지 템플릿은 경험을 편집 한 다음 클릭하여 볼 수 있습니다. 페이지왼쪽 탐색 메뉴에서.

페이지 이름을 클릭하여 편집기에서 페이지 레이아웃을 엽니 다. 위의 예에서집 , 컬렉션 , 비디오 세부 정보 , 검색과프로필페이지를 사용할 수 있습니다.

모바일 앱 템플릿

모바일 앱 템플릿은 기본 모바일 애플리케이션을 만드는 데 사용됩니다. 게시되면 다음을 사용하여 iOS 및 Android 기기에서 액세스할 수 있습니다. 브라이트코브 인게이지™앱 .

모바일 앱 템플릿은 다음과 같은 페이지 레이아웃 템플릿을 제공합니다.

  • 홈
  • 컬렉션
  • 비디오 세부 정보
  • 검색
  • 프로필

웹 템플릿

웹 템플릿은 비디오 중심의 이미지 기반 홈 페이지를 제공합니다. 웹 템플릿은 쉽게 검색 할 수있는 비디오를 지원하고 비디오 설명 위에 마우스를 올려 놓고 콘텐츠의 범주와 하위 범주를 가리 킵니다.

웹 템플릿은 세 가지 페이지 레이아웃 템플릿을 제공합니다.

  • 홈
  • 카테고리
  • 비디오 세부 정보
