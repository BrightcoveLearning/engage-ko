사용자 관리

이 항목에서는 Brightcove Engage를 사용하여 경험을 생성하고 게시 할 수있는 액세스 권한이있는 사용자를 관리하는 방법에 대해 알아 봅니다.
 

사용자 관리 페이지는 경험을 생성하고 게시하기 위해 Brightcove Engage 애플리케이션에 로그인 할 권한이있는 사용자를 관리합니다. 이 기능은 사용자가 만든 모바일 앱 및 웹 경험에 로그인하고 액세스 할 수있는 사용자를 제어하는 데 사용되지 않습니다. 사용자 관리 페이지에 액세스하려면 관리자 역할을 가진 사용자여야 합니다.

사용자 관리 페이지에 액세스하려면 다음과 같이 하십시오.

  1. 브라이트코브 참여에™로그인합니다.
  2. 탐색 헤더에서 톱니바퀴 아이콘 () 를 누른 다음사용자 관리를 수행합니다 .

[사용자 관리] 페이지에 다음이 표시됩니다.

  • 성, 이름
  • 이메일 주소
  • 역할 - 표준 또는 관리자
  • 마지막 로그인-사용자가 애플리케이션에 마지막으로 로그인 한 날짜와 시간입니다.

[사용자 관리] 페이지에서는 다음 작업을 수행할 수 있습니다.

  • 새 사용자 생성
  • 기존 사용자 편집
  • 사용자 삭제
  • 기본 사용자 변경

얼마나 많은 사용자를 가질 수 있습니까?

생성할 수 있는 사용자 수는 보유한 계정의 종류에 따라 다릅니다. 사용자를 더 추가해야 하는 경우 플랜을 더 많은 사용자를 제공하는 플랜으로 업그레이드할 수 있습니다. 이 제한은 언제든지 존재하는 총 사용자 수에만 적용됩니다. 10명의 사용자를 생성하고, 5명의 사용자를 삭제하고, 3명의 사용자를 추가로 만드는 경우 8명의 사용자가 있는 것으로 간주됩니다. 사용자 관리 페이지에는 사용할 수 있는 사용자 수와 생성할 수 있는 추가 사용자 수가 표시됩니다.

추가 사용자 구매

사용자를 더 추가해야 하는 경우 플랜을 더 많은 사용자를 제공하는 플랜으로 업그레이드할 수 있습니다. 추가 사용자 추가에 대한 자세한 내용은 브라이트코브에 문의하십시오.

새 사용자 생성

새 사용자를 생성하려면 다음과 같이 하십시오.

  1. 들어가다이메일 , 이름그리고성사용자의.
  2. 사용자에게 역할을 할당합니다.
    • 관리자 - 사용자는 관리 메뉴()
    • 기준 - 사용자는 관리 메뉴()
  3. 사용자추가를 클릭합니다 .

사용자는 계정에 추가되었음을 알리는 이메일 메시지를 받게 됩니다.

기존 사용자 편집

기존 사용자를 편집하려면 다음과 같이 하십시오.

  1. 편집할 사용자를 클릭합니다. 페이지 상단에 사용자 세부 정보가 표시됩니다.
  2. 사용자 세부 정보를 편집합니다.
    참고: 로그인한 사용자는 편집할 수 없습니다.
  3. 저장을클릭하여 변경 사항을 저장합니다.

사용자 삭제

사용자를 삭제하려면 다음과 같이 하십시오.

  1. 삭제 아이콘 (을) 클릭합니다.) 사용자와 연관되어 있습니다.
  2. 확인을클릭하여 삭제를 확인합니다.

기본 사용자 변경

각 계정의 한 사용자가 기본 사용자로 지정됩니다. 주 사용자는 항상관리자역할을 가지며 브라이트코브 지원에서만 삭제할 수 있습니다.

특정 시점에 주 사용자를 변경해야 하는 경우 새 기본 사용자를 선택한 다음 이 사용자를 이 계정의 기본 관리자로만들기를 클릭합니다 .

기본 사용자 변경을 확인하라는 메시지가 표시됩니다.

새 기본 사용자에게표준역할이 있는 경우 기본 사용자가 될 때관리자역할이 할당됩니다.
