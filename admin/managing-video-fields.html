비디오 필드 관리

이 항목에서는 계정의 비디오 필드를 관리하는 방법에 대해 알아 봅니다.

사용자 지정 메타 데이터를 사용하여 비디오에 추가 메타 데이터 필드를 추가 할 수 있습니다. 사용자 정의 필드는 계정 설정의 일부로 추가됩니다. 사용자 정의 필드를 추가 한 후 필드에 값을 할당 할 수 있습니다. 동영상페이지.

사용자 정의 필드 만들기

사용자 정의 필드를 만들려면 다음 단계를 따르세요.

  1. 브라이트코브 참여에 로그인합니다.
  2. 탐색 헤더에서 톱니바퀴 아이콘 () 그리고비디오 필드 . 관리 메뉴에 액세스하려면 계정 관리자 여야합니다.
  3. 비디오 필드 페이지에는 사용자 정의 필드와 모든 표준 비디오 메타 데이터 필드가 표시됩니다. 새 사용자 정의 필드를 만들기 전에 표준 필드를 검토하십시오. 이렇게하면 작업을 수행 할 수있는 표준 필드가 이미 존재하는 경우 사용자 정의 필드를 만들지 않아도됩니다.
  4. 사용자 지정 필드추가를 클릭합니다 .
  5. 표시 이름을 입력합니다 . 이것은 비디오 속성 페이지의 필드에 대한 레이블로 나타납니다. 표시 이름에 따라 필드에 대한내부 이름이자동으로 생성됩니다. 내부 이름은 변경할 수 있지만 내부 이름에는 공백, 더블바이트 문자 또는 구두점을 포함할 수 없습니다. 예를 들어 표시My Field이름으로 입력하면 내부 이름이 됩니다my_field .
    참고: 사용자 정의 필드 내부 이름으로사용해서는 안 되는 일부 예약어가 있습니다 (표시 이름은 중요하지 않음). 자세한 내용은예약 이름섹션을 참조하십시오.
  6. 선택유형필드를 위해.
    • 끈 -사용자는 최대 1024 개의 1 바이트 문자 (또는 512 개의 2 바이트 문자)까지 필드에 무엇이든 입력 할 수 있습니다.
    • 목록 - 사용자는 값 목록에서 선택합니다. 목록은 150개 이하의 값을 가질 수 있으며 목록의 각 값은 100자 이하여야 합니다.
  7. 설명을 입력합니다 . 설명 필드는 인터페이스에서 힌트로 표시됩니다.
  8. 필드 유형이목록인 경우 쉼표로 구분된 값 목록을 입력한 다음 목록에추가를 클릭합니다 .

    참고 사항:
    1. 값은 따옴표를 포함하지 않아야 합니다 (예: 동물, 새, 물고기).
    2. 값에는 쉼표가 포함될 수 없습니다. 모든 쉼표는 다른 값의 구분 기호로 처리됩니다.
    참고: 사용자 정의 필드 값이 3 자 미만인 경우아니그 가치를 기반으로 동영상을 검색 할 수 있습니다. 예를 들어 부울과 같은 사용자 정의 필드를 생성하려는 경우isfeatureitem , 값 목록을 만듭니다. true그리고false , 아니다yes그리고no ; 후자의 값을 사용하면 ~ 아니다검색을 사용하여 비디오를 찾을 수 있습니다.
  9. 딸깍 하는 소리필드 저장변경 사항을 저장하거나저장 및 추가필드를 저장하고 다른 필드 만들기를 시작합니다.

사용자가 만든 사용자 정의 필드는 즉시 사용할 수 있습니다.  그리고 보일 것입니다 비디오 속성 페이지에서.

최대 10개의 사용자 정의 필드를 생성할 수 있습니다. 10개 이상의 필드가 필요한 경우 더 많은 필드가필요합니까? 를 클릭합니다.비디오 필드 페이지의 링크를 클릭하여 브라이트코브 고객 지원팀에 연락하여 자세한 내용을 요청하십시오. 게시자는 50개의 사용자 정의 필드로 제한됩니다.

목록 값 편집

목록 유형 사용자 정의 필드에 값을 추가할 수 있습니다. 사용자 정의 필드 이름 링크를 클릭하고 값을 추가한 다음 목록에추가를 클릭합니다 . 완료되면 필드저장을 클릭합니다 .

목록 값 삭제

목록 유형 사용자 정의 필드에서 기존 값 제거를 추가할 수 있습니다. 사용자 정의 필드 이름 링크를 클릭하고 삭제 아이콘() 값과 연결된 다음필드 저장 .

목록 값이 제거되면 이전에 해당 값이 할당된 모든 동영상이 값을 유지합니다. 단, 삭제된 값이 있는 영상을 편집하면 값이 지워지고 새로운 값을 선택해야 합니다.

계정에서 사용자 정의 필드 제거

계정에서 사용자 지정 필드를 제거하려면 브라이트코브 지원에문의해야 합니다 . Brightcove 지원에서 사용자 정의 필드를 삭제하도록 설정한 경우 값이 있는 모든 비디오에 대해 수정된 날짜가 변경됩니다.

예약 된 이름

사용자 정의 필드 내부 이름으로사용해서는 안 되는 일부 예약어가 있습니다 (표시 이름은 중요하지 않음). 하나의 일반적인 이름은 범주. 사용자 지정 필드의 내부 이름으로 예약 된 이름을 사용하는 경우 해당 필드를 검색해도 결과가 반환되지 않습니다. 예약 이름을 사용하는 사용자 지정 필드가 이미 있는 경우 해당 필드를제거하고 다른 이름으로 바꾸거나 브라이트코브 지원에 문의하여 지침을 받아야 합니다.
